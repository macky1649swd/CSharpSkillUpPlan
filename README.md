# 🚀 C# / .NET スキルアッププラン（3か月集中コース）

このリポジトリは、**C# と .NET の設計・レビュー力を強化するための3か月計画**です。  
Program Manager・リードエンジニアが「コードレビューで確信を持ってコメントできる」状態を目指します。

---

## 🎯 目的

- C#の言語仕様と .NET の設計思想を体系的に理解する  
- 非同期処理・DI・GC・アーキテクチャを自信を持ってレビューできるようにする  
- 実際のコードを最小限書き、理論を“体感的に理解”する

---

## 🗓️ 計画概要

- **期間**：3か月（12週間）
- **1〜4週**：C#の文法と責務分離  
- **5〜8週**：.NET基盤（GC / Task / DI / EF Core）  
- **9〜12週**：Clean Architectureとレビュー実践  

詳細は 👉 [docs/WEEKLY_PLAN.md](docs/WEEKLY_PLAN.md)
